This is a simple quiz web app built using HTML, CSS, and JavaScript.

## Features
- Random 10-question quizzes  
- Timer system  
- Score tracking  
- Badges (Genius, Smart, Learner)  
- Leaderboard  

## How to use
1. Open `index.html`
2. Enter your name
3. Choose a category
4. Start the quiz
